つなげておぼえる英単語

単語の意味だけでなく、語感とコロケーションのイメージでつかむ。
CEFR-J準拠の単語リストを、自分に合ったフィルターと
カスタムリストで、自分のペースで学習できます。

基本無料 Android
Google Playでダウンロード 機能を見る

// word lists

何が学べるか

CEFR-Jの単語リストをベースに、レベル・条件別に整理された単語を学習できます。

LEVEL A1

基礎レベル

日常生活で頻出する基本単語

LEVEL A2

初級レベル

身近な話題を表現できる単語

LEVEL B1

中級レベル

学校・受験でよく出る単語。AIコロケーション機能対応

LEVEL B2

中上級レベル

より抽象的・学術的な単語。AIコロケーション機能対応

// features

メインコンテンツ

覚えるだけでなく、絞り込んで・並べて・聞いて覚える。多彩な機能が学びを立体的にします。

🔍

複数条件フィルター

CEFRレベル・品詞・既習状態など、複数の条件を組み合わせて単語を絞り込めます。

📝

カスタム単語リスト

自分専用のリストを作って、苦手な単語や試験対策の単語だけをまとめて学習できます。

🔊

ネイティブ読み上げ

端末のネイティブTTS機能で、単語の発音をいつでも確認できます。

🇯🇵

日本語訳付き

CEFR-Jの単語リストに日本語訳を添えて、意味の理解をサポートします。

// ai collocation

コロケーション分布表示（AIコロケーション機能）

単語の「意味」だけでなく「よく一緒に使われる単語」の感覚を、データで補強します。

PAID FEATURE

コロケーション分布

B1以上のCEFR-J単語について、よく一緒に使われる単語の分布を視覚的に確認できます。例文だけでは見えにくい「語の相性」のイメージをつかめます。

◈ 買い切り（追加コンテンツ）

⚠ NOTICE ・コロケーションデータは事前に生成されたものを表示する機能で、利用中にあなたの入力内容がAIへ送信されることはありません。
・分布データは統計的な傾向を示すものであり、すべての用法を網羅するものではありません。

// HOW IT WORKS

AIコロケーション機能のデータは、開発時にAIを使って分析・生成し、サーバーから配信される静的なデータです。
基本機能（単語リスト・フィルター・カスタムリスト・発話など）はすべて無料でお使いいただけます。

// faq

よくある質問

単語の意味だけでなく、語感とコロケーションのイメージでつかむことを目指した英単語学習アプリです。CEFR-Jの単語リストをベースに、フィルターやカスタムリストで自分に合った学習ができます。

はい、単語の学習・フィルター・カスタムリスト作成・発話機能など、基本機能はすべて無料でご利用いただけます。AIコロケーション機能は買い切りの追加コンテンツです。

B1以上のCEFR-J単語について、よく一緒に使われる単語の分布を表示する追加機能です。事前生成済みのデータを表示する仕組みのため、利用中の入力内容がAIに送信されることはありません。

現在はAndroid（Google Play）のみ対応しています。iOS版は今後対応予定です。

Containnect（コンテネクト）が開発・提供しています。「知識をコンテナとしてパッケージ化し、繋げることで学びを拡張する」というコンセプトのもと、インタラクティブな教育プロダクトを開発しています。
